Research Problem

Hybrid Ensemble Implementation

def hybrid_predict(X):
    # Get individual model predictions
    rf_proba = rf_model.predict_proba(X)
    svm_proba = svm_model.predict_proba(X)
    lstm_proba = lstm_model.predict(X.reshape(1, -1))
    
    # Dynamic weighting based on validation F1
    total_weight = rf_f1 + svm_f1 + lstm_f1
    hybrid_proba = (rf_proba*rf_f1 + 
                   svm_proba*svm_f1 + 
                   lstm_proba*lstm_f1) / total_weight
    
    return np.argmax(hybrid_proba, axis=1)

The ensemble combines predictions using F1-score based weighting (RF: 0.94, SVM: 0.92, LSTM: 0.93)
