Multi-Granularity Joint Computing Framework

A modern, structured implementation for Citizen Science Project Management

1. Website Purpose & Users

Primary Users

  • ✔ Development team (5 MIS students)
  • ✔ Instructor (for evaluation)
  • ✔ Client (scientist)
  • ✔ Research team

User Requirements

  • Transparent task and job coordination
  • Multi-level access rights and data visibility
  • Secure and efficient collaboration
  • Tracking contributions and project milestones

2. Functional Requirements

  • Secure User Authentication & Role Management: Users log in/register with role-based access (admin, contributor, evaluator).
  • Project Lifecycle Management: Create, assign, and track projects and tasks with status and deadlines.
  • Document Collaboration: Real-time collaborative editing and version control.
  • Contribution Analytics: Visual dashboards showing user contributions, task completion rates, etc.
  • Notifications & Messaging: Alert users to updates, task changes, and deadlines.
  • Multi-granularity Control: Role-specific visibility for fine-grained control of research data.
